Entrevista

¿Qué cree usted que sea lo más importante para la construcción de una montaña rusa?

Principalmente yo creo que lo más importante para una montaña rusa es la seguridad y que todos los componentes de esta estén bien estructurados y principalmente que esta tenga buenas bases para que no pase nada de mayor problema.

¿Qué materiales utilizaría para construir una montaña rusa?

Si será una montaña real no un prototipo en mi opinión tendría que ser con acero y aleaciones metálicas principalmente, pero si se será un prototipo utilizaría madera como material principal y tal vez tubos PVC y PET reciclado.

¿Qué principio considera el más importante para la construcción de una montaña rusa?

Usaría la física para poder medir la velocidad que esta tendría y cálculo para las pendientes y la inclinación que debe de llevar cada vuelta.

¿Qué consejo nos podría dar para nuestro proyecto?

Como consejo puedo decirles que planeen bien el material a usar, sobre todo las curvas.

¿Cree que los materiales que estamos utilizando sean buenos para nuestra montaña?

Si, son muy resistentes los materiales que están utilizando y de buena calidad.

¿Considera que el prototipo es el adecuado para guiarnos para hacer la montaña rusa?

Si, yo creo que es muy importante el que su prototipo este bien estructurado para que la montaña no tenga ninguna falla.

¿Qué sugiere para la edificación de la montaña?

Que este se empiece hacer ya, para que cualquier falla o problema que se les llegue a presentar lo puedan resolver de manera rápida y con tiempo.
